기술 스택

Frontend 기술

  • React - UI 라이브러리
  • TypeScript - 타입 안정성
  • Vite - 빠른 빌드 도구
  • Monaco Editor - 코드 에디터

Backend 기술

  • Node.js - 런타임 환경
  • Express - 웹 프레임워크
  • MySQL - 데이터베이스
  • Redis - 캐시/세션 관리
  • OpenAI API - AI 문제 생성

시스템 아키텍처

Monorepo 구조

특징

레벨별 학습 경험

레벨 0-1: 블록 코딩 🧩

대상: 프로그래밍 입문자
특징: 드래그 앤 드롭 인터페이스, 시각적 프로그래밍 블록
채점: 블록 구성요소 및 시퀀스 매칭

레벨 2-3: 클로즈 (빈칸 채우기) ✍️

대상: 기본 문법을 배운 학습자
특징: 부분 완성 코드에서 핵심 키워드 채우기
채점: 정답 토큰 일치 검증

레벨 4-5: 코드 에디터 💻

대상: 숙련된 프로그래머
특징: Monaco Editor 기반, 문법 하이라이팅
채점: 테스트케이스 기반 실행 및 판정

채점 시스템

L0-1: 블록 코딩 채점

  • 구성요소(블록) 매칭
  • 블록 시퀀스 순서 검증
  • 즉각적인 시각적 피드백

L2-3: 클로즈 채점

  • 정답 토큰 일치 검증
  • 대소문자/공백 정규화
  • 부분 점수 부여 가능

L4-5: 코드 실행 채점

  • 테스트케이스 기반 판정
  • 안전한 샌드박스 환경
  • 시간/메모리 제한 검증
  • 상세한 피드백 제공

RESTful API 구조

주요 엔드포인트

문제 생성

POST /api/problems/generate

레벨/주제 입력 → AI가 문제 자동 생성

답안 채점

POST /api/problems/grade

사용자 답안 제출 및 레벨별 채점

문제 조회

GET /api/problems/:id

특정 문제의 상세 정보 조회

데이터베이스 설계

주요 테이블

  • users - 사용자 정보
  • problems - 생성된 문제
  • submissions - 제출 답안
  • test_cases - 테스트케이스
  • user_stats - 학습 통계

기술 스택

  • MySQL 8.x - 관계형 DB
  • Redis - 캐시 & 세션
  • 확장 가능한 스키마
  • 최적화된 인덱싱
보안: JWT 인증
성능: Redis 캐싱
안정성: 트랜잭션 관리
