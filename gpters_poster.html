프로젝트 소개

배경
문제 해결의 필요성
주요 문제점:
• 알고리즘 학습 초기 진입 장벽이 높음
• 학습자 수준에 맞는 맞춤형 문제 부족
• 즉각적인 피드백 시스템의 부재
• 단계별 난이도 조절 어려움

해결 필요성:
• 개인화된 학습 경로 제공
• AI 기반 문제 생성 및 채점
• 실시간 학습 분석 및 피드백
해결
AI 기반 학습 시스템
핵심 솔루션:
• OpenAI GPT를 활용한 문제 자동 생성
• 레벨 0-5 단계별 맞춤형 인터페이스
• 블록코딩 → 클로즈 → 코딩의 점진적 학습

주요 특징:
• 사용자 수준에 따른 난이도 자동 조절
• 즉각적인 채점 및 상세 피드백
• 학습 데이터 기반 약점 분석
• 경험치 시스템으로 동기부여
목표
효과적인 학습 경험
목표 사용자:
• 초보자: 블록코딩으로 개념 이해
• 중급자: 클로즈 문제로 실력 향상
• 고급자: 실전 코딩으로 마스터

기대 효과:
• 알고리즘 학습 시간 50% 단축
• 개인 맞춤형 학습 경로 제공
• 체계적인 실력 향상 지원
• 지속적인 학습 동기 부여

주요 기능

📊 레벨별 학습 시스템

  • 레벨 0-1 (입문): 드래그 앤 드롭 블록 코딩으로 알고리즘 개념 학습, 시각적 피드백 제공
  • 레벨 2-3 (중급): 빈칸 채우기 방식으로 코드 구조 이해, 핵심 로직 작성 연습
  • 레벨 4-5 (고급): 실전 코딩 환경에서 완전한 알고리즘 구현, 테스트 케이스 검증

🎯 AI 기반 문제 생성 시스템

  • OpenAI GPT-4 활용: 사용자 레벨과 학습 이력을 분석하여 최적화된 문제 자동 생성
  • 동적 난이도 조절: 정답률과 풀이 시간을 기반으로 실시간 난이도 자동 조정
  • 다양한 알고리즘: 정렬, 탐색, DP, 그래프 등 30+ 주제별 문제 제공
  • 맞춤형 학습: 약점 분석을 통한 집중 학습 문제 추천

✍️ 다단계 학습 인터페이스

  • 블록 코딩: Blockly 기반 시각적 프로그래밍, 초보자도 쉽게 접근
  • 클로즈 문제: 코드의 핵심 부분만 작성하여 효율적 학습
  • Monaco Editor: VS Code와 동일한 에디터로 실전 코딩 환경 제공
  • 실시간 힌트: 막힐 때 단계별 힌트 시스템 지원

📈 지능형 채점 및 분석

  • 자동 채점: 테스트 케이스 기반 즉시 채점, 정확도·효율성 평가
  • 상세 피드백: 오답 분석 및 개선 방향 제시
  • 학습 분석: 레이더 차트로 알고리즘 유형별 강약점 시각화
  • 경험치 시스템: 게임화 요소로 지속적 학습 동기 부여

Backend 기술

Node.js 18+: 비동기 I/O 기반 서버 사이드 JavaScript 런타임
Express 4: 경량 웹 프레임워크, RESTful API 구현
JWT: 토큰 기반 무상태 인증 시스템
Passport.js: 다양한 인증 전략 지원 미들웨어
OpenAI GPT-4: AI 기반 문제 자동 생성 및 채점
